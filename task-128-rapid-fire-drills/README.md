# Task 128 — Rapid-fire drills

**Category:** Cert & interview prep

## Goal
Quick-recall drills on syntax and gotchas across everything covered so far.

**Real-world scenario:** In a live interview or on-call incident,
there's no time to look up `CREATE MASKING POLICY` syntax from
scratch. These 15 drills are for building the muscle memory a
reference lookup can't give you.

## Steps
1. Run `129_rapid_fire_drills.sql` — for each drill, write the SQL from
   memory before looking at the answer beneath it.
2. Work through it for real against your own recall, not by reading
   the answers first.
3. Answer the understanding check below before moving to the next task.

## Understanding check
See the bottom of `129_rapid_fire_drills.sql` — three questions on
which drills exposed real gaps, delegating "has anything changed"
reasoning to Snowflake vs. doing it yourself, and which syntax is
hardest to recall under pressure.
